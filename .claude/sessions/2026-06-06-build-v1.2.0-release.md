## Decisions

- **Sentry token via `ios/sentry.properties`, not env var.** A GUI Xcode archive is
  launched via LaunchServices and doesn't inherit the shell, so an exported
  `SENTRY_AUTH_TOKEN` would silently skip source-map upload. `sentry-cli` reads
  `auth.token` from the properties file in both upload phases. Validated the `org:ci`
  token before archiving. (Step 0 of the runbook was corrected to reflect this.)
- **Throttled Xcode build concurrency** (`IDEBuildOperationMaxNumberOfConcurrentCompileTasks 2`)
  to defeat the recurring `Internal inconsistency error` race rather than retry-looping.
- **Clean prebuild to fix RN prebuilt link errors** rather than hand-patching Pods —
  regenerates a consistent native project + prebuilt artifacts in one step.
